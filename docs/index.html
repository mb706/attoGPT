How To Read It

Click a token to make the model predict the token that should follow it. The two colored bars under every token are the attention weights from head 0 and head 1 for that prediction. Because this model uses top-k sparse attention, each head lights up at most five source tokens.

Click a predicted token, use top 1, or sample to insert the next token and continue generation. To change an existing token, type in the editor and choose a vocabulary suggestion ranked by edit distance and token frequency.
